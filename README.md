## API Reference

#### Get all users

```http
  GET /api/users
```

#### Get user by id

```http
  GET /api/users/:id
```

#### Get all masterclasses

```http
  GET /api/masterclass
```

#### Get c by id

```http
  GET /api/masterclass/:id
```

#### Get masterclass by category

```http
  GET /api/masterclass/:categoryId
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |

#### Get item

```http
  GET /api/items/${id}
```
